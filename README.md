# Template Vue2 Manage

基于 Vue.js 2.x 和 ant-design-vue 组件库且风格统一的用于快速构建中后台系统的脚手架。


## 快速开始

```bash
# 安装
yarn install

# 启动开发服务
yarn dev

# 生产环境打包
yarn build

# 测试环境打包
yarn build:dev
```
